JussKickz Live Inventory Manager
A real-time sneaker inventory management web app built to track and sync sneaker listings across platforms.
Originally built to manage JussKickz, a sneaker reselling business with over 1,000+ customers and $700K+ in revenue, this system provides instant stock updates, SKU monitoring, and data syncing with Google Sheets and Firebase.
Features
Live Syncing: Updates inventory in real-time between the web dashboard and Google Sheets
Product Management: Add, edit, or delete sneaker listings with brand, size, and price fields
Cloud Integration: Firebase backend with Google Sheets API for live data sync
Authentication: Secure admin login for internal management
